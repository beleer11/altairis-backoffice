# Viajes Altairis - Backoffice MVP

Sistema de gestión hotelera para Viajes Altairis, un grupo internacional especializado en distribución hotelera y servicios turísticos B2B.

## Tecnologías Utilizadas

### Backend

- **.NET 9** - Framework principal# Viajes Altairis - Backoffice MVP

Sistema de gestión hotelera para Viajes Altairis, un grupo internacional especializado en distribución hotelera y servicios turísticos B2B.

## Tecnologías Utilizadas

### Backend

- **.NET 9** - Framework principal
- **Entity Framework Core** - ORM para base de datos
- **PostgreSQL** - Base de datos relacional
- **Swagger** - Documentación de API

### Frontend

- **Next.js 14** - Framework React con App Router
- **Tailwind CSS** - Estilos y diseño responsive
- **Recharts** - Visualización de datos y gráficos
- **Axios** - Cliente HTTP

### Infraestructura

- **Docker & Docker Compose** - Contenedorización

## Requisitos Previos

- Docker y Docker Compose instalados
- Git (opcional)

## Instalación y Ejecución

### Con Docker (Recomendado)

````bash
# Clonar el repositorio
git clone <tu-repositorio>

# Levantar todos los servicios
````
